# Tokenomics (APM Token)

APM is the native utility token used for all platform operations.

## Token Utility
- **Market participation:** stake to submit predictions.  
- **Staking:** lock tokens for additional rewards.  
- **Governance:** vote on market and oracle approvals.  
- **Reward payouts:** winners receive tokens based on accuracy.  
- **Liquidity operations:** support exchange and trading.

## Token Distribution

| Sector               | Percentage |

## Supply Governance
- Community governance controls:
  - New market approvals  
  - Oracle integrations  
  - Treasury allocations
